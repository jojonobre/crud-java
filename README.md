# CRUD java

### Projeto desenvolvido para estudos de Java com foco nos conceitos fundamentais da linguagem e na implementação de operações CRUD (Create, Read, Update e Delete).

</div>

--- 

## Objetivo

Este projeto foi criado com o objetivo de praticar:

* Sintaxe básica do Java
* Programação Orientada a Objetos (POO)
* Classes e Objetos
* Encapsulamento
* Listas e coleções
* Manipulação de dados em memória
* Estruturação de aplicações Java
* Operações CRUD

## Funcionalidades

*  Cadastrar produto
*  Listar produto cadastrados
*  Atualizar informações de produto
*  Remover produto(s)
*  Buscar produto por ID

##  Como Executar

### Clone o repositório

```bash
git clone https://github.com/jojonobre/crud-java
```

### Acesse a pasta do projeto

```bash
cd crud-java
```

### Compile o projeto

```bash
mvn compile
```

### Execute a aplicação

```bash
mvn exec:java
```

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos importantes da linguagem Java, incluindo organização de código, modelagem de classes e implementação das operações básicas de um sistema CRUD.
